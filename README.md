# Resume_Analyzer
AI-powered resume analyzer with ATS scoring, job matching, and smart feedback using FastAPI and Streamlit.
🚀 AI Resume Analyzer

An intelligent AI-powered web application that analyzes resumes and compares them with job descriptions to provide a match score, skill insights, and improvement suggestions.

---

📌 Features

- 📄 Upload resume (PDF)
- 🎯 Job description matching
- 📊 Match percentage score
- ✅ Extracted & matched skills
- ❌ Missing skills detection
- ⚡ Fast and user-friendly interface

---

🛠️ Tech Stack

- Frontend: Streamlit
- Backend: FastAPI
- AI/NLP: Python (spaCy / OpenAI / etc.)
- Other: REST APIs

---

🖥️ How to Run Locally

1. Clone the repository

git clone https://github.com/nancysahuofficial21-sudo/Rsume_Analyzer.git
cd resume-analyzer

2. Install dependencies

pip install -r requirements.txt

3. Run backend

uvicorn main:app --reload

4. Run frontend

streamlit run app.py
